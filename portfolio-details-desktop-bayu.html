Project Overview

Aplikasi Desktop Kasir Kedai Kopi ini saya bangun khusus untuk UMKM di daerah Bekasi. Sistem ini memuat data pegawai, data transaksi keluar dan masuk, serta fitur ireport untuk menampilkan laporan keuangan dan transaksi. Aplikasi ini membantu pemilik usaha dalam mengelola operasional kasir secara efisien dan terstruktur.

Berikut tampilan login, menu, transaksi, dan ireport.

Key Features

Manajemen Data Pegawai

Mengelola data pegawai kasir dan staf secara terpusat untuk kemudahan administrasi.

Transaksi Kasir

Mencatat transaksi penjualan dan pembelian secara real-time, baik transaksi masuk maupun keluar.

Laporan iReport

Menampilkan laporan keuangan dan transaksi secara detail menggunakan iReport untuk analisa bisnis.

Login & Menu Navigasi

Fitur login untuk keamanan akses dan menu navigasi yang memudahkan penggunaan aplikasi.
