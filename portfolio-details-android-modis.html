Project Overview

MODIS adalah aplikasi penjualan berbasis Android (Flutter) yang digunakan oleh tim sales untuk melakukan transaksi penjualan di lapangan. Aplikasi ini memiliki dua menu utama untuk proses SO (Sales Order) dengan fitur scan item, ubah harga, serta melihat laporan penjualan secara langsung. Untuk pembayaran, MODIS terintegrasi dengan Xendit, menggunakan database MySQL, dan didukung dashboard web berbasis Laravel untuk monitoring dan analisis data penjualan.

Berikut beberapa tampilan menu utama aplikasi.

Key Features

Sales Order (SO) dengan Scan Item

Memudahkan sales melakukan input pesanan dengan scan barcode, ubah harga, dan kelola item secara fleksibel.

Laporan Penjualan Real-time

Sales dapat langsung melihat laporan penjualan dan performa transaksi dari aplikasi.

Pembayaran Terintegrasi Xendit

Proses pembayaran digital yang aman dan praktis melalui integrasi dengan Xendit.

Dashboard Web Laravel

Monitoring dan analisis data penjualan secara terpusat melalui dashboard web berbasis Laravel.
